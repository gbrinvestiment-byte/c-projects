# Calculadora em C

Projeto simples de calculadora desenvolvido em C para praticar lógica de programação.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Menu interativo

## Tecnologias

- Linguagem C
- GCC
- Terminal

## Objetivo

Praticar:
- estruturas condicionais
- switch case
- operadores matemáticos
- entrada e saída de dados
